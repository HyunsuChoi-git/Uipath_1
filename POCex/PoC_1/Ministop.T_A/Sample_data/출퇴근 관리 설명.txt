[점포] 
1. SA 점착현황 - 전체내역출력
2. 가장 빠른 점착시간이랑 가장 늦은 점출시간
3 점출시간이 -00:00:00 인건 빈칸으로
4. 출퇴근 시간 비교해서 더 빠른것으로 채운다. (해당 Output화면에서)

[비고]
1. 없음 -> 빈칸

[휴가원] 
1. 미니송 - 개인별연차등록 - 일자별연차조회
2. CountIF - 2개 이상있는경우 오류처리
3. 개인별연차등록 N/A는 지우고
4. 연차휴가 -> 연차 // 반차휴가 -> 반차 // 경조사휴가 -> 경조사 // 청원휴가(유급) -> 청원 //외출 -> 외근(외출) // 복귀 ->외근(복귀) // 외근(외출/복귀)
5. 외출일 경우에만 퇴근시간을 18:00로 변경
6. 합쳐가지고 두개 있는것만 오류표시
